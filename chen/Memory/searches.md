# לוג חיפושים — חן

זהו לוג כל החיפושים שביצעתי. לפני כל חיפוש אני בודקת כאן (עם Grep על מילות המפתח) אם כבר חיפשתי משהו דומה ב-30 הימים האחרונים. אחרי כל חיפוש אני מוסיפה כאן entry בפורמט הקבוע.

פורמט entry:

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
